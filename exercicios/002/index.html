Principais formatações

Aqui veremos algumas tag que são semânticas e não semãntica, uma tag semântica é uma tag que possui significado ela tem um objetivo, ja uma tag não semãntica ela so tem forma

NEGRITO / DESTAQUE

Nesta frase temos um termo em negrito usando a tag B (não semântico)

Nesta frase temos um termo de destaque usando a tag STRONG (semântico)

Itálico / Ênfase

Nesta frase temos um termo em Itálico usando a tag I (não semântico)

Nesta frase temos um termo com Ênfase usando a tag EM (semântico)

Marcação de texto

Podemos tambem criar um texto marcado usando a tag MARK (não semântica)

Texto grande e pequeno

Nesta parte temos uma palavra pequena usando a tag SMALL (semântica)

Para palavras maiores não é usado tag hoje em dia, se usa CSS

Texto deletado

para texto deletado ou "irrelevante" podemos usar a tag DEL (semântica)

Texto inserido

para texto inserido ou "adicionado recentemente" podemos usar a tag INS(semântica)

Também tem uma tag que sublinha a palavra que é a tag U (não semântica)

Texto sobrescrito e subscrito

para inserir coisas do tipo x20+3 sobrescrito fica acima da palavra alocada SUP

para inserir coisas do tipo H20 subscrito fica debaixo da palavra alocada usamos a tag SUB
